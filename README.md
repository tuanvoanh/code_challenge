# Code Challenge
API for pharmacy

## Prerequisite
- install nodejs (version 12)
- setup MongoDB

## Installation
- Install all parkage

```bash
npm i
```
- Copy .env.example to .env (can change env variable)
- Start server
```bash
npm run start
npm run dev (to use nodemon)
```
- Testing
```bash
npm run test 
```
